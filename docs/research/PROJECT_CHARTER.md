# Project Charter

## Mission

Advance transparent, community-governed research for mental-health-adjacent foundation models while prioritizing safety, privacy, and clinical boundaries.

## First Milestones

1. Define data governance and contributor review process.
2. Build safety and boundary evaluation sets.
3. Train small adapter experiments on consented, licensed, low-risk data.
4. Publish model and data cards for every release.
5. Invite interdisciplinary review before public model-weight release.

## Open Questions

- What use cases are low-risk enough for an initial release?
- Which datasets have explicit consent and compatible licenses?
- Which safety evals should block release?
- What governance process should approve data additions?
